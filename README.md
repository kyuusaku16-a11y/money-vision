# 🌱 マネービジョン

**未来の資産を、見える化する。**

年齢と収支を入れるだけで、将来の資産推移・資産寿命・目標到達年齢がグラフでわかる無料シミュレーターです。

## 特徴

- **30秒でグラフが出る** — 基本7項目はデフォルト入力済み。スライダーを動かすとリアルタイムに再計算
- **資産寿命がわかる** — 「あなたの資産は何歳まで持つか」を主役指標として表示
- **現実的な計算** — 資産を現金と投資に分けて管理。生活費が足りない分は投資資産を取り崩す、正直なモデル
- **登録不要・データは端末内のみ** — 入力値はブラウザの localStorage に保存。外部送信はありません

## 計算モデル（概要）

- 投資資産にのみ想定利回り（実質利回り）を複利適用
- 現役期間: 収入 − 支出 − 積立 の余剰が現金に貯まる。積立は現金→投資への移動（二重計上しない）
- 退職後: 年金 − 老後支出 で現金が増減。退職年の退職金を加算
- 現金が不足したら投資資産から取り崩す（全期間）
- インフレ・税金は含みません（利回りは実質値として入力）

## 開発

```bash
cd app
python3 -m http.server 8000   # http://localhost:8000/ で起動
npm test                       # ユニットテスト（node --test）
```

Vanilla JS（ES Modules）+ Chart.js。ビルド不要。

## 免責事項

本アプリは簡易シミュレーションであり、投資助言ではありません。計算結果は将来の成果を保証するものではありません。
